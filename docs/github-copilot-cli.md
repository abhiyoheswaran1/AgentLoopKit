# GitHub Copilot CLI

Install GitHub Copilot CLI guidance:

```bash
agentloop install-agent github-copilot-cli
```

The generated file tells Copilot CLI sessions to use the repo harness, task contracts, verification reports, and reviewer handoffs.

AgentLoopKit avoids claiming exact external config conventions when they are uncertain. Link or paste the generated Markdown into the supported location for your setup.
