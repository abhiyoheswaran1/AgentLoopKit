# AgentLoopKit Logo

These are the official AgentLoopKit icon assets used by the README and generated launch visuals.

- `icon.svg` is the source for documentation and screenshots.
- `icon.png`, `icon-192.png`, `icon-256.png`, `icon-512.png`, and `icon-1024.png` are raster exports for surfaces that need PNG.

Use the SVG when possible. It keeps the rounded app icon edge clean and avoids adding a white matte around the icon on light backgrounds.

Do not edit generated README screenshots by hand. Update the HTML source in `docs/assets/readme/` and regenerate screenshots with Playwright.
