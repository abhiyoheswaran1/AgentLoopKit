# Comparison

## Plain AGENTS.md

`AGENTS.md` gives instructions. AgentLoopKit adds artifacts that reviewers can inspect: task contracts, gates, policies, verification reports, summaries, and rollback notes.

## Prompt Collections

Prompt collections improve agent behavior at conversation time. AgentLoopKit keeps the working loop in the repository so it survives across tools and sessions.

## Superpowers

Superpowers is a broader agent methodology and skills system. AgentLoopKit focuses on repo-level engineering loops, task contracts, verification evidence, and handoff artifacts.

They can be used together. AgentLoopKit has no affiliation with Superpowers.
